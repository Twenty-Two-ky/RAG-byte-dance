## 下一个升级阶段是什么
根据 项目重构设计文档.md 的规划，下一阶段是：

Phase 2：LLM Client 升级（LCEL Chain）
目标：用 LangChain 的 LCEL（| 管道语法）替换现有的原始 google-generativeai SDK 调用，创建 core/lc_chain.py。

涉及内容：

新建 core/lc_chain.py，封装 Gemini 模型为 LangChain ChatModel
用 LCEL 组装 Prompt | LLM | OutputParser 链
现有 core/llm_client.py 里的 generate 逻辑迁移到 LCEL 链
价值：

简历上可以写"使用 LangChain LCEL 构建 LLM 调用链"
为后续 Phase 4 ReAct Agent 打基础（Agent 需要 LangChain 的 LLM 接口）

Phase 2: LCEL Chain（LLM 调用层）
